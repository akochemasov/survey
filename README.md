# Опросник

## Разработка
###  Установка зависимостей
```shell
make install
```

### Локальный запуск проекта
```shell
make start
```
Доступен по ссылке http://localhost:8080/

### Сборка проекта
```shell
make build
```

### Запуск тестов
```shell
make test
```

### Запуск линтера
```shell
make lint
```
